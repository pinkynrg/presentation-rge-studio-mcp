# presentation-rge-studio-mcp

Slidev deck: **RGE Studio MCP** — bringing RGE Studio to any AI assistant.

## Develop

```bash
npm install
npm run dev      # http://localhost:3030
```

Edit [`slides.md`](./slides.md). React components live in [`react-components/`](./react-components).

## Deploy

Pushing to `main` builds the deck and publishes it to GitHub Pages via
[`.github/workflows/deploy.yml`](./.github/workflows/deploy.yml).
